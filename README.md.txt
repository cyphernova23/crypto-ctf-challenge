# Base64 Secret Message

## Challenge Overview
An encrypted message was intercepted during an investigation. Your task is to identify the encoding method and recover the hidden flag.

## Category
Cryptography

## Difficulty
Easy

## Learning Objectives
- Understand Base64 encoding
- Learn to identify common encodings
- Practice decoding encoded messages

## Setup Instructions
Open the challenge.txt file and analyze the encoded message. Use appropriate tools to decode it.

## Story
A suspicious message was intercepted from an attacker. Investigators believe the secret flag is hidden using a common encoding technique.

## Flag Format
GCS{...}

## Expected Solution Path
1. Identify that the text is Base64 encoded.
2. Decode the message.
3. Retrieve the flag.

## Learning Outcome
Participants will learn how Base64 encoding works and how to decode encoded messages.
